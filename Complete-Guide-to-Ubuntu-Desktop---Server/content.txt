# Ubuntu Desktop

## Introduction

Ubuntu supports all of your everyday computer demands by providing necessary apps for web browsing, texting, gaming, and content creation, such as Firefox, Chrome, Discord, Steam, and OBS Studio.

## Features

### Key Tools and Apps
- **Web Browsing**: Firefox, Chrome
- **Texting**: Discord
- **Gaming**: Steam
- **Content Creation**: OBS Studio

### Latest Tools & Libraries
- The most recent toolchains, libraries, and apps are included with every Ubuntu release.
- All major IDEs, game creation tools, and AI/ML software are available.

### Security & Privacy
- Ubuntu is the safe option for those who value data protection.
- Prioritizes user privacy and system integrity through frequent updates and integrated security safeguards.
